# 更新日志

## 0.1.0-rc.2

### 登录与会话

- 优化 Web 端冷启动恢复登录状态时的过渡表现，减少页面切换带来的闪烁感。

## 0.1.0-rc.1

### 登录与会话

- 修复 Web 端关闭浏览器后再次打开仍需重新登录的问题，现在会在会话有效期内自动恢复登录状态。

## 0.1.0-alpha.2

### 登录与会话

- 优化 Web 端与桌面端的登录续签体验，减少使用过程中频繁重新登录。
- 将服务端 access token 有效期调整为 24 小时。
- 会话失效时会自动清理不可用登录状态，减少切换到失效账号或停留在伪登录状态的情况。

### 桌面端

- 禁用登录窗口的原生账号自动填充与历史用户名提示。
- 更新应用图标与品牌视觉资源。

### 网页端

- 更新网页端图标与品牌视觉资源。
- 导航卡片支持右键编辑和删除。

## 0.1.0-alpha.1

### 桌面端

- 发布基于 Tauri、React 和本地 SQLite 的桌面客户端。
- 提供导航面板、账户切换、应用设置与自动更新能力。
- 支持与 Navix 服务端进行版本校验、会话化同步和图标传输。

### 服务端

- 发布基于 Rust、Axum、SQLx 和 SQLite 的服务端。
- 提供管理员初始化、管理员创建用户、登录、刷新令牌、登录状态校验和账户管理能力。
- 提供兼容性检查、同步会话、图标上传下载和管理员接口。
- 支持结构化日志、链路追踪字段和内嵌网页端静态资源分发。

### 交付与部署

- 提供 Docker 镜像构建与发布流程。
- 支持通过 GitHub Release 发布桌面端、服务端和 Docker 镜像。

### 说明

- 当前版本为 Alpha 阶段，接口与行为可能继续迭代。

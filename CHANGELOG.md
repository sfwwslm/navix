# 更新日志

格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，并遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)。

## 0.2.0 - 2026-04-22

### Added

- Web 端 Launchpad 新增搜索能力，并支持使用内置搜索引擎发起检索，提升启动页内的内容查找与快速搜索体验。

### Changed

- 调整桌面端路由结构，移除 generouted 并重建路由组织，为后续跨平台客户端演进清理结构基础。
- 将桌面端工作区统一更名为 client，统一项目内客户端命名，便于后续多平台扩展。
- 补充 Apache-2.0 与 MIT 许可证文件，完善仓库分发与开源许可信息。

### Fixed

- 修复 Web 端 Launchpad 在构建场景下活动分组状态同步异常的问题，避免当前分组显示与实际状态不一致。
- 修复桌面端拖拽排序后 `sort_order` 未及时同步的问题，避免编辑后条目或分组跳回顶部。
- 修复桌面端退出过程中数据库路径相关错误日志，减少关闭应用时的误导性报错。
- 修复系统托盘图标错误提示文案，减少托盘异常时的误导信息。

## 0.1.0 - 2026-04-18

### Added

- Web 端控制中心支持删除已保存账号，便于清理不再使用的登录记录。

### Changed

- 重构 Web 端控制中心弹窗布局，优化信息组织与操作区域分布。
- 统一 frontend lint 与类型检查脚本的工作区调用方式，减少命令执行差异。

### Fixed

- 修复 Web 端退出登录后仍可能恢复陈旧会话的问题，避免重新进入无效登录状态。
- 修复 Web 端控制中心瞬时提示消息不会自动清理的问题，减少残留提示干扰。
- 禁用桌面端输入框的浏览器自动补全行为，提升桌面应用登录与编辑体验。
- 修复桌面端在元数据获取失败后，手动上传图标无法正常显示的问题。

## 0.1.0-rc.2 - 2026-03-30

### Changed

- 优化 Web 端冷启动恢复登录状态时的过渡表现，减少页面切换带来的闪烁感。

## 0.1.0-rc.1 - 2026-03-30

### Fixed

- 修复 Web 端关闭浏览器后再次打开仍需重新登录的问题，现在会在会话有效期内自动恢复登录状态。

## 0.1.0-alpha.2 - 2026-03-29

### Added

- 网页端导航卡片支持右键编辑和删除。

### Changed

- 优化 Web 端与桌面端的登录续签体验，减少使用过程中频繁重新登录。
- 将服务端 access token 有效期调整为 24 小时。
- 会话失效时自动清理不可用登录状态，减少切换到失效账号或停留在伪登录状态的情况。
- 禁用桌面端登录窗口的原生账号自动填充与历史用户名提示。
- 更新桌面端与网页端的应用图标及品牌视觉资源。

## 0.1.0-alpha.1 - 2026-03-24

### Added

- 发布基于 Tauri、React 和本地 SQLite 的桌面客户端，提供导航面板、账户切换、应用设置与自动更新能力。
- 支持桌面端与 Navix 服务端进行版本校验、会话化同步和图标传输。
- 发布基于 Rust、Axum、SQLx 和 SQLite 的服务端，提供用户管理、登录鉴权、兼容性检查及同步会话能力。
- 服务端支持结构化日志、链路追踪字段和内嵌网页端静态资源分发。
- 提供 Docker 镜像构建与发布流程，支持通过 GitHub Release 发布多端资源。
